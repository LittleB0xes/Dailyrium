# Dailyrium
Dailyrium is a new attempt to make a roguelike with Rust. At the beginning, I decided to use bracket_lib, but, there is always *but*, I didn't feel comfortable with it...

So I moved to Tetra game engine...

A *blog* of my advancement is viewable [here](https://github.com/LittleB0xes/Dailyrium/tree/main/Unexpected_Journey/README.md)

**Warning**
This code is perhaps, certainly, mostly awful for a real Rustecean, but I am still in the learning phase (for a looooong time, obviously)...
